# Library-Management-System

<h2>Description</h2>
A robust Library Management System built in Java with a graphical user interface using Swing. This project enables users to add, view, issue, and return books with real-time updates. Designed for efficient demonstration in interviews and educational presentations.

<h2>Features</h2>
<u>
<li><b>Add New Books:</b> Enter book title and author via dialog.</li>

<li><b>Display All Books:</b> View the complete list of booksâ€”status shown as "Issued" or "Available".</li>

<li><b>Issue Book:</b> Select and issue a book, updating its status.</li>

<li><b>Return Book:</b> Easily return issued books and update inventory.</li>

<li><b>Responsive GUI:</b> User-friendly interface designed with Java Swing.</li>
</u>
<h2>Technologies Used</h2>
<u>
<li>Java SE 8+</li>

<li>Swing Framework for GUI</li>

<li>Collections (ArrayList) for data storage</li>
</ul>
